##  Быстрый запуск

Инструкция предназначена для операционной системы **Windows**.

### 1. Подготовка окружения

Убедитесь, что у вас установлен **Python 3.x**. Откройте командную строку (CMD) в корневой папке проекта (`C:\projects\pygame-top-down-shooter`):

```cmd
# Создание виртуального окружения
python -m venv venv

# Активация окружения
venv\Scripts\activate

```

> **Примечание:** Если вы используете PowerShell и получили ошибку выполнения скриптов, выполните: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process`, а затем повторите активацию.

### 2. Установка зависимостей

После активации (когда слева в консоли появилось `(venv)`) установите необходимые библиотеки:

```cmd
pip install -r requirements.txt

```

*Если файл requirements.txt отсутствует, установите Pygame вручную:*

```cmd
pip install pygame

```

### 3. Запуск игры

Для начала игры выполните команду:

```cmd
python main.py

```
